```text
주어진 문자열 중 k개만큼 빼서 가장 큰 수를 출력하는 문제이다.
단, 정렬을 해서는 안된다.
그리디한 문제이다.
리스트에 저장해두고 하나씩 뽑아쓰면 된다.
```
```python3
def solution(number, k):
    answer=[]
    for i in number:
        if not answer:
            answer.append(i)
            continue
            
        while answer[-1]<i and k>0:
            answer.pop()
            k-=1
            if not answer or k<=0:
                break
                
        answer.append(i)
        
        if len(answer)==len(number)-k:
            break
            
    return ''.join(answer)
```
