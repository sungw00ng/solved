### dfs
```python3
def solution(numbers, target):
    def dfs(idx,hap):
        if idx==len(numbers):
            if hap==target:
                return 1
            else:
                return 0
        return dfs(idx+1,hap+numbers[idx])+dfs(idx+1,hap-numbers[idx])
    return dfs(0,0)
```
