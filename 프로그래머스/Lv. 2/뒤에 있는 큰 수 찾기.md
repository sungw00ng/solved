## 뒤에 있는 큰 수 찾기
https://school.programmers.co.kr/learn/courses/30/lessons/154539 <br>

```text
스택 선형 문제이고, 비슷한 유형이 많아서
쉽게 풀 수 있었다.
```

```python3
def solution(numbers):
    n=len(numbers)
    answer=[-1]*n
    stack=[]

    for i in range(n):
        while stack and numbers[stack[-1]]<numbers[i]:
            idx=stack.pop()
            answer[idx]=numbers[i]
        stack.append(i)

    return answer
```
